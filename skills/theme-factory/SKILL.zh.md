# 主题工厂技能

此技能提供精心策划的专业字体和颜色主题集合，每个主题都有精心选择的调色板和字体配对。一旦选择主题，可以应用于任何artifact。

## 目的

要对演示文稿幻灯片应用一致、专业的样式，请使用此技能。每个主题包括：
- 具有十六进制代码的连贯调色板
- 标题和正文文本的互补字体配对
- 适合不同背景和受众的独特视觉识别

## 使用说明

要对幻灯片或其他artifact应用样式：

1. **显示主题展示**：显示`theme-showcase.pdf`文件，让用户可以视觉上看到所有可用主题。不要对其进行任何修改；只为查看而显示文件。
2. **询问选择**：询问要将哪个主题应用于幻灯片
3. **等待选择**：获取关于所选主题的明确确认
4. **应用主题**：一旦选择主题，将所选主题的颜色和字体应用到幻灯片/artifact

## 可用主题

以下10个主题可用，每个在`theme-showcase.pdf`中展示：

1. **海洋深处** - 专业且平静的海事主题
2. **日落大道** - 温暖鲜艳的日落色彩
3. **森林树冠** - 自然和接地的大地色调
4. **现代极简主义** - 清洁和当代的灰度
5. **黄金时间** - 丰富温暖的秋季调色板
6. **北极霜冻** - 凉爽清新的冬季主题
7. **沙漠玫瑰** - 柔和精致的尘埃色调
8. **科技创新** - 大胆现代的技术美学
9. **植物园** - 新鲜有机的花园色彩
10. **午夜银河** - 戏剧性和宇宙的深色调

## 主题详细信息

每个主题在`themes/`目录中定义，包含完整规范，包括：
- 具有十六进制代码的连贯调色板
- 标题和正文文本的互补字体配对
- 适合不同背景和受众的独特视觉识别

## 应用过程

选择首选主题后：
1. 从`themes/`目录读取相应主题文件
2. 在整个幻灯片中一致地应用指定的颜色和字体
3. 确保适当的对比度和可读性
4. 在所有幻灯片中保持主题的视觉识别

## 创建您自己的主题
要处理现有主题都不适合artifact的情况，请创建自定义主题。基于提供的输入，生成类似于上面主题的新主题。给主题一个类似的名称，描述字体/颜色组合代表什么。使用提供的任何基本描述来选择适当的颜色/字体。生成主题后，显示以供审查和验证。之后，按上述描述应用主题。
